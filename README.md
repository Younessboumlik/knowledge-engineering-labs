# Knowledge Engineering Labs

> A comprehensive collection of Jupyter notebooks focused on **Fuzzy Logic** and **Knowledge Engineering** concepts with practical implementations and Machine Learning integration.

## üìã Overview

This repository contains a series of practical workshops (TPs - Travaux Pratiques) and exercises that explore fuzzy logic systems, fuzzy control, and their integration with machine learning techniques. The materials are designed for students and practitioners interested in knowledge engineering and intelligent systems.

## üìö Contents

### 1. **TP1 & TP2: Fuzzy Logic** (`tp1ANDtp2.ipynb`)
A comprehensive tutorial covering fundamental and advanced fuzzy logic concepts:

- **TP1: Fuzzy Logic Fundamentals**
  - Exercise 1: Introduction to fuzzy sets and membership functions
  - Exercise 2: Fuzzy operations and rules
  - Exercise 3: Fuzzy inference with improved resolution (0.1 step for smoother curves)
  - Exercise 4: Advanced fuzzy system applications

- **TP2: Fuzzy Logic and Machine Learning Integration**
  - Combining fuzzy logic with machine learning techniques
  - Hybrid intelligent systems
  - Practical applications and case studies

### 2. **Exercise 1: Fuzzy Control System** (`exo1.ipynb`)
A detailed implementation of a **Temperature-Based Speed Regulation** system using fuzzy logic:

- Fuzzy logic control system design
- Temperature monitoring and speed control
- Real-world control system simulation
- Visualization of fuzzy sets and control surfaces

### 3. **Workshop: Industrial Machine Failure Prediction** (`atelier.ipynb`)
A practical workshop on **Fuzzy Prediction of Industrial Machine Failure Risk**:

- **Fuzzy Modeling**: Building fuzzy systems for risk assessment
- **Machine Learning**: Integrating ML techniques for predictive maintenance
- Real-world industrial application
- Risk prediction and decision-making systems

## üîß Prerequisites

### Required Software
- Python 3.7 or higher
- Jupyter Notebook or JupyterLab

### Required Python Libraries
Install the necessary dependencies using pip:

```bash
pip install numpy pandas matplotlib scikit-fuzzy scikit-learn
```

Or create a virtual environment and install the requirements:

```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate

# Install dependencies
pip install numpy pandas matplotlib scikit-fuzzy scikit-learn jupyter
```

## üöÄ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Younessboumlik/knowledge-engineering-labs.git
   cd knowledge-engineering-labs
   ```

2. **Install dependencies**
   ```bash
   pip install numpy pandas matplotlib scikit-fuzzy scikit-learn jupyter
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. **Open and run the notebooks**
   - Start with `tp1ANDtp2.ipynb` for fundamental concepts
   - Proceed to `exo1.ipynb` for practical control system implementation
   - Explore `atelier.ipynb` for real-world industrial applications

## üìñ Topics Covered

- **Machine Learning Integration**: Hybrid systems, predictive modeling, classification
